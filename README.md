
# Insurance Management System (Java)

A comprehensive JavaFX-based Insurance Management System with MySQL database integration for managing customers, policies, and premium calculations.

##  Features

###  Customer Management
- Add new customers with personal details (Name, Surname, Age, Address)
- Unique customer ID generation
- Customer information validation
- Database persistence

###  Policy Management
- Create insurance policies (Life, Health, Auto)
- Set coverage amounts and calculate premiums
- Policy-customer relationship management
- Real-time premium calculations

###  Premium Calculation
- Automated premium calculation based on policy type:
  - **Life Insurance**: 5% of coverage amount
  - **Health Insurance**: 8% of coverage amount
  - **Auto Insurance**: 10% of coverage amount
- Dynamic premium updates

###  User Interface
- Modern JavaFX GUI
- Form validation and error handling
- Policy table view with sorting
- Responsive layout design
- Professional styling

###  Database Features
- MySQL database integration
- Foreign key relationships
- Prepared statements for security
- Connection pooling
- Data persistence

##  Technologies Used

- **Language**: Java 8+
- **GUI Framework**: JavaFX
- **Database**: MySQL
- **Build Tool**: Maven/Gradle
- **IDE**: NetBeans/IntelliJ IDEA
- **Version Control**: Git
